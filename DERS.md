# Ders 1: Degisken, olay ve fonksiyon

Bu uygulamada uc temel kavram var:

1. `let count = 0;`
   VBA'daki `Dim count As Integer` satirinin modern JavaScript karsiligidir.

2. `addEventListener("click", ...)`
   Visual Basic'teki `Button_Click` olayinin karsiligidir.

3. `function showCount()`
   VBA'daki `Sub ShowCount()` prosedurunun karsiligidir.

## Alistirma

`app.js` dosyasindaki `count = count + 5;` satirini
`count = count + 5;` olarak degistirin. Dugmeye bastiginizda sayacin
beser beser arttigini gozlemleyin.
